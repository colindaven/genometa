IGB 6.2.2 release notes

IGB 6.2.2 incorporates additional formats and data sets, UI improvements, memory management, and other bugfixes.

IGB 6.2.2 requires Java 6 or higher.

  • Enhanced DAS/1 support for Ensembl (and others).
  • External (UCSC) browser tab
  • Localization
  • GUI bugfixes and enhancements (threading, zooming)
  • Preferences bugfixes and enhancements (data sources, linkouts)
  • BAM support
  • USeq support
  • DAS/2 server: Serving of partial sequences
  • DAS/2 server: Loading of compressed (.gz, .zip) files
  • DAS/2 server: Security bugfix
  • DAS/2 server: 2bit, USeq support


IGB is part of the GenoViz project. Source code for IGB is released under the Common Public License, v1.0 an OSI approved open source license. IGB uses other open source software packages which are covered by their own open source licenses.

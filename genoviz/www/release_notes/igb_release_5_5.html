IGB 5.5 release notes

IGB 5.5 is an incremental release that incorporates UI improvements, memory management, and other bugfixes.
IGB 5.5 requires Java version 1.5 or higher. Releases after IGB 5.5 will require Java 6.
  • Image export (print to file) for many formats, such as: PostScript, PDF, EMF, SVF, Flash SWF, as well as GIF, PNG, JPG and PPM.
  • Image export (print to file) for Sliced View.
  • Clearer and more robust "Add Server" page.
  • New IGB icon and splash screen.
  • Major performance improvement in Heatmaps for mid- to large- size graphs.
  • More efficient Wiggle format representation, using an estimated ~30% less memory.
  • Several parser bug fixes.
  • Finite timeouts for remote resources (DAS servers, etc.)
  • Closing resource leaks.
  • Case sensitivity improvements for data sets.
  • Default shortcuts.
  • Additional unification for DAS/1.
  • New Genoviz FASTQ parser.


IGB is part of the GenoViz project. Source code for IGB is released under the Common Public License, v1.0 an OSI approved open source license. IGB uses other open source software packages which are covered by their own open source licenses.

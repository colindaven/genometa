IGB 5 release notes.

IGB 5 is a major new release that emphasizes the use of the DAS/2 protocol.

IGB 5.01 now requires Java version 1.5 or higher.
  • DAS/2 has replaced QuickLoad as the primary way to access remote data.
    DAS/2 allows access to wider selection of annotations than QuickLoad, more loading options, and better performance.
    You can revert back to using QuickLoad instead of DAS/2 via a toggle in the "Data Sources" preferences tab.

  • Genome selection is now done via "Pick Genome" button in top ribbon.

  • Sequence selection is now done via Sequence drop-down in top ribbon.

  • IGB now keeps much better track of settings from previous sessions.

  • Annotation Browser tab is turned off by default.
    Instead use DAS/2 search via "Name Search" subtab in "Data Access".

  • Pivot View tab is now deprecated and is turned off by default.

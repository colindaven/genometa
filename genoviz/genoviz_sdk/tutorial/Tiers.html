TieredNeoMaps

If you see this text, your browser apparently doesn't support Java!

TieredNeoMaps are an extension of NeoMap that allows for categories of data to be grouped in tiers. Tiers are actually a type of glyph called MapTierGlyphs, which expand as necessary as more child glyphs are added to them. In this way, the display can be divided on the fly into areas of appropriate size for each category of data.

To learn more about how to use a TieredNeoMap, please refer to the TieredNeoMapDemo.

GenoViz Vertical NeoMap Demo

Here we demonstrate a vertically oriented map. Both maps below were built with the exact same API calls, except for different constructors: 

  map = new NeoMap(map.HORIZONTAL);
  map = new NeoMap(map.VERTICAL);
This demo also illustrates the usage of a hairline glyph, also called a Shadow glyph. Zooming along the primary axis will be centered along the point indicated by the hairline.

CAVEAT: Some of the glyphs included in the GenoViz works best with horizontal map. Those descended from DirectedGlyph works best for both horizontal and vertical map.

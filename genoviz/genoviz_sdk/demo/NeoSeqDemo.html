GenoViz SDK NeoSeq Demo

If you see this text, your browser apparently doesn't support Java!

This demo exercises many of the methods in the NeoMapI API. Notice when resizing the window that the NeoSeq compensates, and only displays rows in round increments of 10 or 20.   Take a look at the source, which is considerably longer than most of the tutorials.   However, notice that most of this code is in adding lots of menus and tying those menus to NeoSeqI API calls.   The part that sets up the map as it initially appears is still fairly small.   For more details on the API calls this applet demonstrates, take a look at the JavaDocs for NeoSeqI and NeoWidgetI.

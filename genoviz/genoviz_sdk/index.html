GenoViz Software Development Kit

Welcome to the GenoViz Software Development Kit

The GenoViz SDK is a set of Java-based components for building sophisticated bioinformatics data-visualization applications.

The GenoViz SDK consists of a number of "widgets" that can be used to visualize many types of bioinformatics data. The following widgets are available in GenoViz:

  • NeoMap for annotated linear maps (physical, genetic, sequence, etc.)
  • TieredNeoMap for annotated linear maps with several groups of data.
  • NeoAssembler for sequence assemblies and alignments
  • NeoSeq for sequences at residue-letter resolution
  • NeoQualler for sequencing quality scores
  • NeoTracer for DNA sequencing traces
Refer to the Tutorial below for more information on these widgets.

Documentation

Javadocs are constructed from information inside the source code. To construct the javadocs, execute the command "ant docs" from the "genoviz_sdk" directory.

Before using the tutorial or demos, it is necessary to compile the source code using the ant task: "ant demos".
